Spann Table

Description

This table combines the principles of form and function into a single object. I set out to create an object that is beautiful from all angles while still serving its purpose as an entryway table. The mahogany tabletop provides ample area for smaller items while the lower shelf is designed for papers.

I challenged myself with the overall design and construction methods. I lasercut the pattern of different holes, inspired by the San Francisco De Young museumâ€™s copper facade. The table legs create compound angles as they span outward from the center of the table. I used a milling machine and TIG welder to achieve the exact angles and precise welds necessary for the stability of the table.

Info

Built using laser cutter, TIG welder, manual milling machine, lathe, coldsaw, table saw, orbital planer
Steel tube and lasercut sheet, stained mahogany | 46in x 11in x 38in
